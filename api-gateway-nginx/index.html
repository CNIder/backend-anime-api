🎉 Bem-vindo à Anime API!

Explora os microserviços disponíveis e seus endpoints:

Users Service

  • GET /users – Listar todos os utilizadores
  • GET /users/{user_id} – Obter utilizador pelo ID
  • POST /users – Criar novo utilizador
  • PUT /users/{user_id} – Atualizar utilizador
  • DELETE /users/{user_id} – Remover utilizador
  • 📄 Documentação Swagger

Anime Service

  • GET /anime – Listar animes
  • GET /anime/{anime_id} – Obter anime pelo ID
  • 📄 Documentação Swagger

Ratings Service

  • GET /rating – Listar todas as avaliações
  • POST /rating – Criar nova avaliação
  • GET /rating/user/{user_id} – Avaliações de um utilizador
  • GET /rating/anime/{anime_id} – Avaliações de um anime
  • PUT /rating/{rating_id} – Atualizar avaliação
  • DELETE /rating/{rating_id} – Remover avaliação
  • 📄 Documentação Swagger
